Welcome to documentation of the QGIS plugin SciPy Filters!
==========================================================

The QGIS plugin **SciPy Filters** provides access to `SciPy <https://scipy.org/>`_ filters via the processing toolbox. 

Python users get extra tools for working in the QGIS Python console.
